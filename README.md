## 🚀 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Git

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Developer-Yogeshwaran/portfolio.git
   cd Portfolio-master
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
   The app will open in your browser at `http://localhost:3000`

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📖 Usage

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs tests
- `npm eject` - Ejects from Create React App (one-way operation)

### Customization

To customize this portfolio for your own use:

1. **Update personal information** in `src/components/About/AboutCard.js`
2. **Modify GitHub username** in `src/components/About/Github.js`
3. **Update portfolio links** in `src/components/About/ProjectLinks.js`
4. **Edit projects** in `src/components/Projects/Projects.js`
5. **Customize colors** in CSS files (e.g., `src/App.css`, `src/style.css`)
6. **Replace placeholder images** in `src/Assets/`
